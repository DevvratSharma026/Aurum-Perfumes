# AURUM — Maison de Parfum

A cinematic luxury perfume brand website **solely built by AI — Claude (Anthropic)** through vibe coding. Every line of code, every design decision, and every animation was generated by Claude Sonnet. Created as a portfolio piece demonstrating what's possible with AI-assisted development.

## Built By

This entire project — architecture, GLSL shaders, animations, design system, and deployment config — was generated through natural language prompts to **Claude by Anthropic**. Zero manual code was written.

## Live Demo

[aurum-perfumes-flame.vercel.app](https://aurum-perfumes-flame.vercel.app)

## Built With

- **React + Vite** — component architecture and blazing fast builds
- **Three.js + React Three Fiber** — custom GLSL fluid shader with real-time cursor displacement
- **GSAP + ScrollTrigger** — scroll-driven animations, text reveals, section transitions
- **Lenis** — smooth inertia scrolling
- **Kling 2.5** — AI-generated cinematic perfume pour video
- **Midjourney** — AI-generated product bottle photography

## Features

- Custom GLSL fluid shader with 6-layer FBM noise and mouse distortion — runs full-screen as a fixed background
- Scroll-driven shader color transitions across three scent worlds (Noir → Or → Blanc)
- AI-generated video section scrubbed on scroll entry
- Cinematic product panels with scroll-triggered bottle and text animations
- Full product overlay — fragrance pyramid, quality metrics, quantity selector, add to cart
- Custom gold cursor with smooth lag
- Code-split lazy loading — 16kb initial bundle
- Gzip + Brotli compression on all assets
- Content Security Policy headers

## Performance

- Initial JS bundle: 16kb (gzipped)
- Code split into 7 async chunks
- Single unified RAF loop — no competing animation frames
- Shader capped at 4 FBM iterations (-30% GPU vs standard 6)
- DPR capped at 2x
- Tab visibility culling — shader pauses when tab is hidden
- React.memo on all static components
